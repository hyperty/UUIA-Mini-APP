# UUIA-Mini-APP
UUIA服务框架 —— 小程序客户端已经发布

# 快速上手
1. 下载小程序客户端源码
   ```bash 
   git clone https://github.com/uuia/UUIA-Mini-APP.git
   ```

2. 访问 [UUIA 主仓库](https://github.com/uuia/UUIA) 及 [UUIA 开放平台](https://uuia.cheelem.com/dev)阅读使用说明。

3. 修改`/uuia/config.js`内的配置项，完成部署
